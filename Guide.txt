Goal  
  
Build a small Product Dashboard in React + TailwindCSS + shadcn/ui using the DummyJSON API.  
  
Features to Implement  
  
1. Layout & UI  
   
Responsive dashboard layout (sidebar + header + main content).  
  
Use TailwindCSS for styling and at least 2 shadcn/ui components (e.g., Card, Table, Dialog, Button, Skeleton).  
  
1. Data Table (Products List)  
   
Fetch products from:  
  
GET https://dummyjson.com/products?limit=10&skip=0  
  
Show in a table with columns: Title, Price, Category, Stock.  
  
Add Next / Previous buttons for pagination.  
  
Add a search input to filter products by name.  
  
Show loading (skeleton) and error states.  
  
Handle empty search results gracefully. 
 
  
1. CRUD Operations  

Add Product:  
  
Show a button “Add Product” → opens a Dialog form.  
  
Refresh the product list after success.  
  
Edit Product:  
  
Each row should have an “Edit” button.  
  
Opens dialog pre-filled with product data.  
  
Refresh product list after success.  
  
Delete Product:  
  
Each row should have a “Delete” button.  
  
Refresh table after success.  

  
1. React Query Integration  
  
Use TanStack Query (React Query) for:  
  
Data fetching with caching  
  
Pagination + filters  
  
Mutations (POST, PUT, DELETE)  
  
Auto refetch after mutation  
  
Bonus (Optional)  
  
Add a dropdown filter for category.  
  
Show total count (total from API response) and current page info.  
  
Add artificial delay (?delay=1000) to demonstrate proper loading states.  
  
Expectations  
  
Clean, responsive UI (desktop + mobile).  
  
Good component structure (split UI and logic).  
  
Proper handling of loading, error, empty states.  
  
Working CRUD actions with list refresh.  
  
Code pushed to GitHub repo with a short README.md (setup + usage).  


  
README.md with:  
  
Setup instructions  
  
Libraries used  
  
Short explanation of your approach
